* Overview

ExploringEmu is a CLI application demonstrating LocalForage capabilities through a retro gaming emulator inventory management system. It showcases CRUD operations with persistent storage using LocalForage's IndexedDB backend.

* Quick Start

#+BEGIN_SRC bash
# Setup and run
just setup
#+END_SRC

* CLI Reference

#+BEGIN_SRC bash
# Basic CRUD operations
./src/index.js add --name "SNES9x" --platform "SNES" --version "1.62"
./src/index.js list
./src/index.js get --id "snes9x"
./src/index.js update --id "snes9x" --version "1.63"
./src/index.js delete --id "snes9x"
./src/index.js clear
#+END_SRC

* Features

- Create, Read, Update, Delete operations
- Persistent storage via LocalForage
- Verbose logging with multiple levels
- SOLID design principles
- Unix philosophy compliance
